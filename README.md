# AI安全开放社区

欢迎加入**AI安全开放社区（Open Community for AI Safety China）**。

我们希望打造一个开放、自由、平等的 AI 安全社区，促进知识分享、学术交流和行业发展，确保 AI 造福人类社会。

目前正在开展的活动包括线上讲座和讨论会。后续会推出更多共创项目，敬请期待。

请关注社区的官方账号以获取最新动态
